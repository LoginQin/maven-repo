# maven-repo
My personal maven repository.

## Usage
pom.xml:
```xml
    <repositories>
        <repository>
            <id>qwei-maven-repo</id>
            <url>https://raw.githubusercontent.com/LoginQin/maven-repo/master</url>
        </repository>
    </repositories>
```
